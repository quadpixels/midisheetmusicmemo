《爱背谱／MidiSheetMusic Memo》

在线说明：http://edgeofmap.com/blog/saltine-midisheetmusic/

这是一个用于背五线谱，并为背谱的目标——演奏进行准备的一个应用程序。这个应用程序是在 Madhav Vaidyanathan 所开发的MidiSheetMusic的基础上开发的。

这个应用程序中也受到了《基础钢琴练习法》一书的启发。

【帮助屏幕】

在应用程序第一次启动到乐谱显示界面和问答界面时，会显示帮助屏幕。

帮助屏幕显示完一次后就不会再出现了。如果想要让它们再出现，就点击屏幕上面的复原按钮。

【关于掌握程度】

乐谱中由很多小块（一个声部里的一个小节）组成，每个小块有一个掌握程度，在问答模式完后会进行结算。根据掌握程度的高低，这些小块会被归于几摞中。

没有进行过问答的小块处于最左上角的摞中。每次问答答对或答错就会沿着箭头移动（答对就沿绿箭头移动，答错则沿红箭头移动）。

 

每个圈里的数字表示这摞中的小块数。

如果答错了在填空模式中出现的小块，则这个小块沿红箭头移动。答对了则沿绿箭头移动。移动到了绿圈中的话，就表示这个小块己经完全掌握了。

每一摞中的小块的掌握程度（从红到绿映射为0到1）加起来就是整个曲谱的掌握程度（记为XP），其最大值就是一共有的小块数。

在XP值的右边的柱状图表示每一摞中的小块数的相对多寡，颜色与每一摞的颜色对应。

你会发现图中有两个表示完全掌握了的绿圈。设计这样两个一样的绿圈是因为：

  • •如果你之前己经对这个谱子比较熟悉了，那么也应该很容易能一连答对三次，然后把谱中的小块移到上面的绿圈中，或者 

  • •如果你之前对这个谱子不是太熟因而答错了一两次，就需要多做几次练习巩固记忆。在巩固完了之后，也要接连答对三次，才算彻底掌握了。 

如果想到达上面的绿圈，则要连续答对三次。如果想到达下面的绿圈，则最少要答对五次，且最后三次必须连续答对才行。

根据《基础钢琴练习法》，如果能把背谱背下来，就表示在演奏时出错的卡壳的几率就会小。

以上的掌握程度图能告诉你对这个谱子的大概掌握程度。如果想详细地深入到每一个小节，就可以打开下一节所描述的覆盖图层。

【关于合并成／分解成两个声部】

合并成两个声部是MidiSheetMusic中原有的功能，也移植了过来。

有的曲谱包含多于两个声部（比如钢琴的两个声部加上人声），可以根据音高将这些声部合并成两个。

在合并完后虽然音符还是原来的那些音符，但排版和原先可以相去甚远，看起来就是不同的谱子。

因此，在计算掌握程度时，合并之后的谱子和合并前的谱子就会当作不同的谱子分别计算。

【关于覆盖图层】

把曲谱上拉到底部工具条出现后，点击底部工具条的第三个按钮，就可以打开或选择覆盖图层。覆盖图层会在每个小块上画一个进度条，是用来显示对应每个小块的填空模式统计信息的，就是一个介于0和1之间的数值。进度条越满，则相对应的数值就越高。举例如下：

 

图中所示第一声部的第六小节的回答正确率是大约50％。

统计指标有绿色、红色或蓝色的（这三种颜色会随所使用的配色方案而变化）。

绿色的统计指标是正面的，也就是越高越好，表示掌握程度越高。正面的指标有以下几个：

  • •回答正确率 

  • •掌握程度 

红色的统计指标与绿色相反，是负面的，越低越好。负面的指标有以下几个：

  • •反应时间 

  • •正确作答反应时间 

蓝色的统计指标是中性的，只是个客观存在，并不反应掌握程度。中性指标只有以下一个：

  • •出现次数 

除了这三种有颜色的之外，还有一种统计指标是「在最后五次问答中是否答对」。譬如说：

 

第11小节和第12小节在问答中出现过两次。在这两次中，第11小节最后一次答错了，倒数第二次答对了；第12小节两次都答对了。

【关于填空模式的统计信息】

填空模式的统计信息有两种：完成时间和正确率。

无论在进行填空模式时选定了哪些声部，也无论是否选择将多声部乐谱合并成两个声部或将单声部乐谱分解成两个声部，当次的完成时间和正确率都会添加到之前总共的记录之后而不会分开记录。这样做是因为如果对每一种选择的组合都分开记录的话可能性就会太多，因为一个含有N个声部的曲谱会有至少2^N种组合。

【配色方案的妙用】

配色方案会在文件列表中反映出来，可以当作彩色书签用。譬如可以将觉得难的谱子用某一种颜色标记，就可一目了然。

【为什么要做这个应用程序】

我在自学练琴的时侯看到了《基础钢琴练习法》这本书，发现书中将练琴的各阶段障碍梳理得清楚，比如要能从曲子中途指哪弹哪、分小节慢慢打磨困难的部分等。逐个击破后则就会渐渐有能力自己录视频与人交流或在朋友面前露一手，很有成就感。而背谱就是其中很关键的一步。

这个应用程序是为了让背谱的过程更有意思，让进步的过程更明晰而存在的。

【为什么背谱很重要】

「如果一个人己经能对着在众人或对着摄影机演奏一个曲子，那无论他是有意去背谱也好，还是无意也好，一定是己经记住了这首曲子的。因为只有记住了之后，才能有余力顾及演奏这曲子的技术要求并演奏出来」。（《基础钢琴练习法》第104页）

背谱是音乐工作流程中的重要一步，当尝试背谱之后，很多其它步骤会变简单。在《基础钢琴练习法》中提到了其它几个使你一步一步达到自如演奏的步骤：

图像记忆／声音记忆／肌肉记忆-->断点续弹-->脑中演奏-->在琴上无错自如弹奏

具体来说，这个应用程序中的「填空模式」是一种图像记忆方法，挖掉空格的方法则和断点续弹类似。

【吾谁与归】

我是想和自学钢琴过程中经历过困难挫折和喜悦的人分享这个应用程序的。如果从程序的技术面的限制（排版能力之类）来说，MidiSheetMusic的能力所限就是这个应用程序的能力所限，至少我个人目前能看到的巴赫二部是这个程序足够胜任的。又由于巴赫二部在《基础钢琴练习法》作为例子出现过，所以这个应用程序应该能伴随阅读这本书的过程，对初学者来说该是好长一段时间了吧。
